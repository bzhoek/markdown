Extended from [example](http://www.raywenderlich.com/50151/text-kit-tutorial).

### Paragraph support
* Headers up to 4 #
* Code blocks with tab

### Character support
* Code with backtick `
* Bold with *
* Underline with _
* Strikethrough with -

## BUGS
* Initial load of text with newline repeats paragraph formatting on next line.
